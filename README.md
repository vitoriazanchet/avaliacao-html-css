# Trabalho Individual de Front-End Essencial

### 🎓 Exercício Individual - Residência em TIC/Software Serratec (2026)

Este projeto foi desenvolvido como um exercício prático para a disciplina de **Front-End Essencial** no programa de Residência em Software do **Serratec**. O objetivo é aplicar conceitos de estrutura `HTML` e estilo `CSS`.

---

## 🗂️ Organização do Projeto

* **Divisão de pastas:** 
Foram realizados 3 exercícios com propóstas diferentes e estão separados pelas pastas `exercicio-01`, `exercicio-02` e `exercicio-03`, podendo conter:
    **css**: pasta obrigatória, pois torna a revisão e ajuste do estilo do projeto muito mais célere. E nela está o arquivo `styles.css` que é o arquivo de estilo global.
    **img**: pasta contendo os arquivos de imagens.
    **pages**: pasta contendo as páginas da aplicação.
    **index.htlm**: arquivo obrigatório na raiz do projeto, ele basicamente é a estrutura, o que conecta todos os elementos.
    
---

## 🛠️ Estrutura do Projeto

* **HTML (Estrutura):** Define a marcação do conteúdo, como textos, links e imagens.

* **CSS (Estilo):** Responsável pela aparência, layout, cores e responsividade, frequentemente organizado em pastas e arquivos como css, styles ou assets.

---

## 🚀 Como Visualizar

Os principais métodos de visulização, são:

* **Abrir diretamente no navegador:** Encontre o arquivo `index.html` na pasta do exercício que deseja visulaizar, clique com o botão direito e selecione o navegador de sua preferência.

* **VS Code + Live Server (Recomendado):** 
1. Abra a pasta do projeto no VS Code.
2. Vá em Extensões (Ctrl+Shift+X) e instale o Live Server (caso não tenha).
3. Clique com o botão direito no arquivo HTML e escolha "Open with Live Server".
4. O site abrirá e se atualizará a cada salvamento (Ctrl+S).
